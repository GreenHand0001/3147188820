# 项目简介

## 核心功能
- 基于YOLOv8的水下目标检测系统，支持5类目标检测（eel、fish、human、jellyfish、lobster）
- 提供完整源码、1500张标注数据集、训练好的模型及训练结果
- 支持多种输入源：单张图片、图片文件夹、视频文件、本地摄像头
- 界面功能：实时显示目标位置、目标总数、置信度等信息
- 自定义设置：置信度阈值、IOU阈值、标签显示选项、原图显示选项
- 批量检测功能：支持批量处理并直接查看所有检测结果
- 检测结果保存功能

## 适用场景
- 水下生物研究与监测
- 水下环境安全监控
- 海洋生物多样性调查
- 水下机器人视觉系统
- 水下目标识别与分析

## 技术特点
- 开发环境：Pycharm+Anaconda或VSCode+Anaconda
- 技术栈：Python 3.9、OpenCV-Python、PyQt5、Ultralytics、PyTorch 1.9
- 用户界面：基于PyQt5开发的图形界面，支持直观操作
- 模型性能：已训练完成，配置环境后可直接使用
- 数据集规模：1500张标注图片，涵盖5类水下目标
